Project: Chest X-ray Classification with Convolutional Neural Networks (CNNs)

This project explores the application of Convolutional Neural Networks (CNNs) for classifying chest X-ray images into two categories: pneumonia and healthy.

Tools and Technologies:
Programming Language: Python
Deep Learning Library: TensorFlow
NumPy (numerical computations with arrays)
Pandas (data manipulation and analysis)
Matplotlib/Seaborn (data visualization)

Methodology:
Data Acquisition
Data Preprocessing
Model Building
Model Training
Model Evaluation
